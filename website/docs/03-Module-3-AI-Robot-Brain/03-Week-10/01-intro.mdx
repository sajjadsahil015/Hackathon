## 1. The Nav2 Stack Architecture

Nav2 is the industry standard for mobile robotics navigation.
*   **Planner (Global)**: Dijkstra/A* algorithm to find path from A to B.
*   **Controller (Local)**: Follows the path while avoiding dynamic obstacles (people, pets).
*   **Costmaps**: 2D grid representing safe/unsafe areas.

## 2. Costmap Layers

1.  **Static Layer**: Walls (from SLAM map).
2.  **Obstacle Layer**: Real-time Lidar data.
3.  **Inflation Layer**: Padding around obstacles to prevent collision.
