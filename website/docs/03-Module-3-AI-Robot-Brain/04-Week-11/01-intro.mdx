## 1. State Machines vs Behavior Trees

*   **Finite State Machines (FSM)**: Rigid states (Idle -> Moving -> Error). Hard to scale.
*   **Behavior Trees (BT)**: Modular, hierarchical logic (Sequence, Fallback, Parallel). Used in Halo AI and Nav2.

### BT Nodes
*   **Action**: Do something (Move, Speak).
*   **Condition**: Check something (IsBatteryLow?).
*   **Control**: Logic flow (Sequence: Do A, then B).
